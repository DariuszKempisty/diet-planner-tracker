# Diet Planner Tracker - Kreator Posiłków

Aplikacja do generowania przepisów i precyzyjnego śledzenia wartości odżywczych (witaminy, minerały, kalorie).

## 🚀 Funkcje aplikacji
* **Inteligentny Kreator Posiłków:** Generowanie przepisów dopasowanych do Twojego zapotrzebowania.
* **Zaawansowany Kalkulator:** Precyzyjne obliczanie kaloryczności oraz pełnego profilu mikro- i makroskładników (w tym żelazo, magnez, witaminy itp.).
* **Dopasowanie pod Cele:** Automatyczne skalowanie gramatury składników, aby posiłek idealnie wpisywał się w dzienny limit użytkownika.
